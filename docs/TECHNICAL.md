# Documentación Técnica - CRM con Clientes y Notas

## Información General

**Aplicación:** CRM con gestión de clientes y notas  
**Stack Tecnológico:** HTML + JavaScript  
**Repositorio:** https://github.com/luisernestosalas/app-goj0o  
**URL de Producción:** https://app-goj0o-dva8qhb01-luisernestosalas-2775s-projects.vercel.app  

## Descripción del Sistema

Sistema CRM básico desarrollado con tecnologías frontend nativas que permite la gestión de clientes y sus respectivas notas asociadas.

## Arquitectura Técnica

### Frontend
- **HTML5:** Estructura y marcado semántico
- **JavaScript Vanilla:** Lógica de negocio y manipulación del DOM
- **CSS:** Estilos y presentación visual

### Deployment
- **Plataforma:** Vercel
- **Tipo:** Static Site Deployment
- **CI/CD:** Integración automática con repositorio GitHub

## Funcionalidades Principales

### Gestión de Clientes
- Registro de nuevos clientes
- Visualización de lista de clientes
- Edición de información de clientes
- Eliminación de clientes

### Gestión de Notas
- Creación de notas asociadas a clientes
- Visualización de notas por cliente
- Edición de notas existentes
- Eliminación de notas
