# Manual de Usuario - CRM Simple

## ¿Qué es esta aplicación?

Esta es una aplicación web para gestionar clientes y sus notas. Te permite:
- Agregar nuevos clientes
- Ver la lista de todos tus clientes
- Añadir notas a cada cliente
- Buscar clientes específicos

## Cómo acceder

Ingresa a: https://app-goj0o-dva8qhb01-luisernestosalas-2775s-projects.vercel.app

## Funciones principales

### 1. Agregar un nuevo cliente

1. Busca el botón **"Agregar Cliente"**
2. Completa los datos del cliente:
   - Nombre completo
   - Email
   - Teléfono
   - Empresa (opcional)
3. Haz clic en **"Guardar"**

### 2. Ver lista de clientes

- Todos los clientes aparecen en la pantalla principal
- Cada cliente muestra:
  - Nombre
  - Email
  - Teléfono
  - Empresa
  - Fecha de registro

### 3. Agregar notas a un cliente

1. Busca el cliente en la lista
2. Haz clic en **"Ver detalles"** o **"Agregar nota"**
3. Escribe tu nota en el campo de texto
