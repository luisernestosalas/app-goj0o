# CRM con Clientes y Notas

Una aplicación CRM simple y eficiente para gestionar clientes y sus notas asociadas, desarrollada con tecnologías web nativas.

## 🚀 Demo en Vivo

Accede a la aplicación: [https://app-goj0o-dva8qhb01-luisernestosalas-2775s-projects.vercel.app](https://app-goj0o-dva8qhb01-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- ✅ **Gestión de Clientes**: Crear, editar y eliminar clientes
- 📝 **Sistema de Notas**: Asociar múltiples notas a cada cliente
- 🔍 **Búsqueda y Filtrado**: Encontrar clientes rápidamente
- 💾 **Persistencia Local**: Los datos se guardan en el navegador
- 📱 **Diseño Responsivo**: Compatible con dispositivos móviles
- ⚡ **Rendimiento Optimizado**: Aplicación ligera y rápida
